Trying Hugo

Mar 8, 2016 by John Spickes

I’m giving Hugo a try.

Why?

  • I want my files. I like them to be here, on my machine, editable offline, going with me wherever I go.
  • I like Markdown.
    • I like my files to be in a format I can easily read and edit.
    • I like being able to easily move my content elsewhere when I change my mind. And I will. Markdown is easy to read and convert.
    • I like using my own editor. I use vim all the time at work, and I’m fast with it, and I don’t want to be forced to use someone else’s editor.
  • I’m just that kind of nerdy.

I do need to figure some things out, though.

  • Where will I host the statically-generated pages?
    • I could run my own light http server here at home. That would give me maximum control. I might even be able to host it straight from my NAS.
    • GitHub Pages is an option.
